## 字符串的初始化规则
~~~c
char arr[5] = {'0'};
// 结果: arr = {'0', '\0', '\0', '\0', '\0'}
//       只有第一个是'0'，其余自动填充'\0'

char arr[5] = {'a', 'b'};
// 结果: arr = {'a', 'b', '\0', '\0', '\0'}
//       只初始化前2个，其余自动填充'\0'

char arr[5] = {0};
// 结果: arr = {'\0', '\0', '\0', '\0', '\0'}
//       全部初始化为'\0' (数值0)
~~~
**因此初始化一般为0或者不初始化**
